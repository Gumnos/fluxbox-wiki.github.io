Wer schon immer mal wissen wollte wie man bei den Fenster Menus oder Toolbar diese "runden" Ecken hinbekommt fügt eine der folgenden Zeilen in sein style file ei:

menu.roundCorners: TopRight TopLeft BottomRight BottomLeft

window.roundCorners: TopRight TopLeft BottomRight BottomLeft

toolbar.shaped: true|false

Es müssen nicht alle Optionen gesetzt werden, wer zB. nur oben abgerundete Ecken haben will trägt nur TopRight TopLeft ein. Bei der Toolbar kann nur ein Argument gestzt werden: true = runde Ecken false = keine

Bei borderless Windows werden die Fenster selbst abgerundet.
