Qu'est ce que fluxMenu?

Fluxmenu est un éditeur de menu pour fluxbox qui permet de le modifier facilement avec une belle interface graphique. Il est encore en développement mais contient déjà la majorité des fonctionnalitées :) Si vous avez des questions, contactez le développeur, Zan sur #fluxbox (en anglais).

fluxmenu est développé en python et nécessite pygtk 2.6 et libglade. Python doit évidemment être installé sur le système (2.4 minimum).

Où le télécharger ?

Vous le trouverez sur la page de fluxMenu. Ici le lien direct sur la version actuelle.

Décompresser l'archive avec:

tar -zxvf fluxMenu-24.tar.gz

et lancer fluxMenu: python fluxMenu.py

Pour les utilisateurs chevronés, vous pouvez aussi obtenir la version SVN en tapant:

svn checkout svn://svn.berlios.de/fluxmenu/trunk fluxmenu && cd fluxmenu

Et lancez-le avec: python fluxMenu.py

Je conseille de faire un chmod a+x fluxMenu.py pour pouvoir le lancer en tapant simplement: ./fluxMenu.py

Comment l'installer ?

Pour le moment il n'y a pas d'installateur. Lancez-le simplement comme expliqué ci-dessus.

Comment l'utiliser ?

Lancez-le et il va charger votre menu actuel. Sélectionnez les éléments à éditer, cliquez sur les boutons d'ajout de sous-menu, etc... Quand vous cliquez sur le bouton "save" (sauvegarder), le menu est mis à jour, il suffit ensuite de faire un clic droit sur le bureau pour voir ce que ca donne.

Puis-je aider ?

Bien sur, l'aide est toujours la vienvenue :) J'ai déjà quelques patch/idées reçues par email. N'hésitez pas à m'envoyer des emails pour discuter de fluxmenu mon email sur la page officielle ou venez discuter sur #fluxbox.
