Como hacer que las ventanas no tengan decoraciones

Hay dos maneras de lograr esto. La mas fácil es agregando la siguiente entrada en el archivo de configuracion ~/.fluxbox/keys :

  Mod4 t :ToggleDecor

Puedes usar ese o elegir otra combinación de teclas (la tecla de windows + t en este caso)

Ahora tienes que recargar la configuración con:

  menu fluxbox -> recargar configuración (reload config)

o usando una señal desde la linea de comando de esta manera:

 kill -s usr2 `pgrep fluxbox`

Después de recargar la combinación de teclas ya esta lista para usarse. El proxímo paso es ejecutar la aplicación que queres que no tenga borde.

Haz click con el boton derecho sobre el titulo de la ventana para mostrar el menu. Elije 'Recordar' y marca las opciones 'Decoración' y 'salvar al cerrar' Usa la combinación de teclas para dejar la ventana sin borde, y cierra la. Al hacerlo la configuración se salvara en el archivo ~/.fluxbox/apps. De ahora en mas todas las ventanas del mismo programa se ejecutaran sin bordes, y puede usar la misma combinación para devolverle temporalmente el borde

De esta manera la configuración se aplicara a todas las ventanas del mismo programa. Si tu quieres, por ejemplo que aterm con irssi corriendo en el no tengan borde, pero el resto de las ventanas de aterm tengan una decoración normal, necesitas ponerle un nombre especifico a esta instancia de aterm. La mayoria de las terminales usan el parametro "-name" para esto:

   $> aterm -name irssiterm -e irssi

Aplicandole la combinación de teclas y salvando la configuracion crear una nueva entrada en el archivo ~/.fluxbox/apps similar a la siguiente:

   [app] (irssiterm)  
      [Deco]   {NONE}
   [end]   

Esto solo se aplicara a la aplicación llamada irssiterm

La segunda forma es un poco mas complicada y una parte hay que hacerlo mientras fluxbox no esta corriendo.

Abrí la aplicación que quieres correr sin decoraciones, luego abre una terminal y ejecuta:

  xprop WM_CLASS|cut -d \" -f 2

Haz click en la aplicación. Esto te dirá el nombre para usar en el archivo de apps. Recuerdalo o escribelo porque lo estaremos usando

Cierra fluxbox y edita el archivo ~/.fluxbox/apps y agrega las siguientes lineas

  [app] (nombre de la aplicación que te dije que recordaras)
     [Deco]    {NONE}
  [end]

Guarda el archivo e inicia fluxbox. Las preferencias sera usadas y cada vez que ejecutes esa aplicación, aparecerá sin decoraciones
