Em sistemas conhecidos como unix-like, $PATH é uma variável de ambiente que contém todos os caminhos que seu terminal vai procurar para executar os programas. Para descobrir o que tem no seu $PATH basta abrir um terminal e digitar:

echo $PATH

Você vai ver algo assim:

 /usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games

Isso pode ser muito longo. Para adicionar algo temporariamente para o seu $PATH digite:

export $PATH=$PATH:/usr/local/games/bin

E isso deve adicionar /usr/local/games/bin para o seu $PATH. Para uma solução permanente, você deve adicionar isso para o seu arquivo shellrc. Para usuários BASH vai ser algo como ~/.bashrc e ZSH vai ser ~/.zshrc ou algo parecido. Você ainda pode considerar adicionar essas coisas para ~/.profile
