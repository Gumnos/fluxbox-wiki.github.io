Instalação

Para debian e distribuições baseadas em deb:

apt-get install fbdesk

No Fedora:

yum install fbdesk

No Gentoo

emerge fbdesk

Compilando o código fonte:

./configure (--help   to check all options)
make
make check (optional)
su -
make install

Dependencias

O fbdesk só precisa do libpng.

Configuração

Bem, o fbdesk mostra o ícone padrão em 64 64, nós podemos clicar com o botão direito no ícone e seguir para propriedades

Exit FbDesk

Normalmente o fbdesk detecta automaticamente o estilo que está sendo usado, mas nós podemos acelerar isso adicionando o seguinte comando para ~/.fluxbox/startup :

fbdesk -rc ~/.fluxbox/init &

E nós podemos adicionar as seguintes configurações para ~/.fluxbox/init :

fbdesk.iconFile:   (default: ~/.fluxbox/fbdesk.icons)
fbdesk.font:  (default: fixed)
fbdesk.textColor:    (default: black)
fbdesk.snapX:  (default: 5)
fbdesk.snapY:  (default: 5)
fbdesk.doubleClickInterval:  (default: 200)
fbdesk.textPlacement:   (default: Bottom)
fbdesk.iconAlpha:  (default: 255)
fbdesk.textAlpha:  (default: 0)
# Possible values for text placement: Left, Right, Top and Bottom
